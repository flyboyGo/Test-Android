2、滑动轮播图片

   两种无限循环轮播的实现方式∶
   1、欺骗适配器
      在适配器中将getCount()的值设置为无限大

   2、构造数据源
      有五张图片，实现无限循环。在viewpager中设置7个view，第一个为五张图片的最后—张，第七张为五张图片的第一张。
      图片顺序如下数字∶
               4-0-1-2-3-4-0

      0-1-2-3-4为正常的五个图片; 4, 0为添加的两个图片view

   滑动的顺序∶进入页面显示0图片，向右滑动到0时,将0页设置为0，则可以继续向右滑动。同理当向左滑动到4时，将4页设置为4。
